# CSS Labels

CSS Labels is a lightweight, performance-focused library for rendering large numbers of HTML labels with built-in intersection handling. It was created to support scenes with many thousands of overlapping labels while keeping the API small and the rendering model predictable.

It powers label rendering in  [🪐 Cosmograph](https://cosmograph.app), the fastest web-based network graph rendering framework.

## Why CSS Labels

- Zero-dependency runtime package.
- Built for scale, including large sets of intersecting labels.
- Built-in overlap resolution with weighted label priority.
- Stable visibility decisions to reduce flicker between frames.
- Reuses DOM nodes by label `id` instead of recreating them every draw.
- Uses cached and estimated label measurements to avoid unnecessary layout work.
- Supports plain text, custom CSS, and trusted HTML / multi-line labels.
- Works both as a batch renderer (`LabelRenderer`) and as individual labels (`CssLabel`).
- Optional style injection, exported stylesheet, click handling, and wheel event forwarding for interactive canvases and graphs.

## Quick Start

Install the package:

```bash
npm install @interacta/css-labels
```

Create HTML div element and [render labels](?path=/story/quick-start--labels):

```javascript
import { LabelRenderer } from '@interacta/css-labels'

const div = document.querySelector('#labels')
const renderer = new LabelRenderer(div)

renderer.setLabels([
  { id: 'monster', text: '👾', x: 100, y: 50, opacity: 1 },
  { id: 'alien', text: '👽', x: 50, y: 150, opacity: 1 },
  { id: 'ufo', text: '🛸', x: 150, y: 150, opacity: 1 },
])
renderer.draw()
```

Or create [single Css label](?path=/story/quick-start--single-label):

```javascript
import { CssLabel } from '@interacta/css-labels'
const div = document.querySelector('#labels')

const label = new CssLabel(div, '🐣')
label.setPosition(100, 110)
label.setVisibility(true)
label.setOpacity(1)
label.draw()
```

## Styling

By default, CSS Labels injects its base styles automatically. If you use `dontInjectStyles`, import the bundled stylesheet instead:

```javascript
import '@interacta/css-labels/styles.css'
```

The label appearance can be customized with CSS variables, either globally on `:root` or on a specific labels container. See the [CSS variables story](?path=/story/quick-start--css-variables) for a live example.

Available variables:

- `--css-label-background-color`: label background fill.
- `--css-label-color`: label text color.
- `--css-label-border`: label border.
- `--css-label-box-shadow`: label shadow.
- `--css-label-filter`: visual filter applied to each label.
- `--css-label-border-radius`: label corner radius.
- `--css-label-font-weight`: default text weight.
- `--css-label-transition`: transition used for label state changes.
- `--css-label-cursor`: cursor shown when hovering labels.
- `--css-label-max-width`: maximum label width.
- `--css-label-pointer-events`: default pointer events behavior for each label.

Example:

```css
#labels {
  --css-label-background-color: rgba(118, 224, 196, 0.16);
  --css-label-color: #d6fff4;
  --css-label-border: 1px solid rgba(118, 224, 196, 0.35);
  --css-label-box-shadow: 0 10px 18px rgba(0, 0, 0, 0.22);
  --css-label-filter: drop-shadow(0 10px 18px rgba(0, 0, 0, 0.28));
  --css-label-border-radius: 999px;
  --css-label-font-weight: 700;
  --css-label-transition: opacity 180ms ease, transform 180ms ease;
  --css-label-cursor: pointer;
  --css-label-max-width: 220px;
  --css-label-pointer-events: auto;
}
```
